# WIS2

WIS2 is a web-based information system designed for managing educational courses. The project consists of a Java Spring Boot backend and a React + TypeScript frontend.

## Features
- User authentication and role-based access (admin, teacher, student, supervisor)
- Course management (creation, enrollment, grading, term registration)
- Room and schedule management
- Notifications/messaging system
